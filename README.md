✨ Что такое Mirage
Mirage — это P2P-мессенджер, построенный вокруг приватности по умолчанию.

🔐	Сообщения видите только вы и собеседник. Сквозное шифрование на Ed25519 + X25519 + ChaCha20-Poly1305. Серверы не имеют доступа к содержимому — они лишь помогают клиентам найти друг друга.
🪪	Никаких номеров и email. Учётная запись — это криптографическая пара ключей, созданная на вашем устройстве.
🛡️	Обфусцированный трафик. QUIC поверх UDP с TLS 1.3 — спроектирован так, чтобы быть неотличимым от обычного веб-трафика для DPI-фильтров.
🌐	Mesh-архитектура. Когда прямое P2P-соединение невозможно, сообщения проходят через релейные узлы в зашифрованном виде. Релей видит только маршрутизацию, не содержимое.
⚠️ Alpha-релиз. Mirage находится на ранней стадии. Используйте его для тестирования и знакомства, но не как основной канал для критичных коммуникаций.

🛠 Установка
<details> <summary><b>🪟 Windows</b></summary>
Запустите Mirage-Setup-X.Y.Z.exe.
Если SmartScreen покажет предупреждение об «unknown publisher» — нажмите «Подробнее» → «Выполнить в любом случае». Сборка пока не подписана сертификатом code-signing (планируется в ближайших релизах).
Следуйте инструкциям установщика.
</details> <details> <summary><b>🍎 macOS</b></summary>
Откройте .dmg и перетащите Mirage в папку Applications.
При первом запуске macOS может сказать, что приложение от неизвестного разработчика. Откройте Системные настройки → Конфиденциальность и безопасность, найдите пункт Mirage и нажмите «Открыть».
</details>
🔄 Автоматические обновления
После установки Mirage сам проверяет наличие новых версий через этот репозиторий и предлагает обновиться. Достаточно нажать «Перезапустить» в уведомлении внутри приложения — никаких ручных скачиваний.

Управлять поведением апдейтера можно в Настройках → Обновления.

🔍 Проверка целостности
К каждому релизу прилагается файл SHA256SUMS.txt с контрольными суммами. Проверьте загруженный файл перед установкой:

Windows (PowerShell):

Get-FileHash Mirage-Setup-X.Y.Z.exe -Algorithm SHA256
macOS / Linux:

shasum -a 256 Mirage-X.Y.Z-arm64.dmg
Полученное значение должно совпадать со строкой из SHA256SUMS.txt.
